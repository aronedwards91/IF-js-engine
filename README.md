# IF-js-engine
Internals for a basic interactive fiction engine


#### Special Types

###### Interaction
either a string returned to the reader, or trigger (starts with *)

###### Parsable Description
[[imug01:taken:false==and a mug]]

###### Number States
*++ : increases by 1
*+2 : increase by 2
*-2 : decrease by 2
*-- : decrease by 1



#### Problems
 optional / status driven exits
